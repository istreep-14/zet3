**Badge** — tiny uppercase mono labels for staff roles, closer status, and bonuses.

```jsx
<Badge variant="bartender">Bartender</Badge>
<Badge variant="closer">Closer</Badge>
<Badge variant="bonus" pill>+$24</Badge>
```

Variants: `neutral`, `bartender` (blue), `server` (gold), `support` (purple), `closer` (red), `bonus` (green). Set `pill` for the fully-rounded bonus marker. These are intentionally micro — ~7.7px — so they sit inline beside names and values without competing.
